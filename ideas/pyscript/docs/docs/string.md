# **String**

Transform the value to different **Cases**.

## Set A Value

```js
const value = plugin.str("I'm Twisted !!!");
```

## **Transform** The Value

---

- **Lower**: `value.lower;`
- **Upper**: `value.upper;`
- **Title**: `value.title;`
- **Pascal**: `value.pascal;`
- **Camel**: `value.camel;`
- **Slug**: `value.slug;`
- **Snake**: `value.snake;`
- **Cut**: (Integer) `value.cut(size);`
